# Getting Started

## Dev instructions

1. Install dependencies (Node 16):

```bash
npm install

2. Run the development server:

```bash
npm run dev
```

4. Open [http://localhost:3002](http://localhost:3002) with your browser to see the result.

5. The site auto-updates as you edit the file.


## Build the website

1. Run build to verify that the build passes

```bash
npm run build
```

2. Git push to the "main" branch these changes, The Netlify hook takes care of the deployment
